## 什么是 Anet

Anet 是一个 .NET 通用库，适用于 .NET 6 及以上版本。

> A .NET Core Common Lib, Framework and Boilerplate.

它的取名正是来自于这个定义的前面四个字母：ANET。Anet 的宗旨是使 .NET 项目开发变得简单和快速。
